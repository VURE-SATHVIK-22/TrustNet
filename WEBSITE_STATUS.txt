================================================================================
                    TO ENABLE BACKEND (ML FEATURES)
================================================================================

YOU NEED TO INSTALL PYTHON FIRST:

Option 1: Install from Python.org
   1. Go to: https://www.python.org/downloads/
   2. Download Python 3.11 or 3.12 (recommended)
   3. Run the installer
   4. ✓ CHECK "Add Python to PATH" during installation
   5. Complete installation
   6. Restart your computer

Option 2: Install from Microsoft Store
   1. Open Microsoft Store
   2. Search for "Python 3.11" or "Python 3.12"
   3. Click Install
   4. Wait for installation to complete

After Installing Python:
   1. Open Command Prompt or PowerShell
   2. Type: python --version
   3. You should see: Python 3.x.x
   4. Then navigate to trustnet folder
   5. Run: python start-ml-backend.py
   6. Backend will start on http://localhost:8000
